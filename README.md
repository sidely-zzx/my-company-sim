
# 项目简介
这个项目是一个模拟外包公司的web游戏，后期可以打包为electron桌面应用到steam上发行

## 主要玩法
玩家扮演一个外包公司的老板，可以接甲方公司发布的项目外包和人力外包。然后去招聘员工，也可以先招聘员工再去接外包项目。通过平衡收入和支出赚取差价。在初期开发先开发表单数据和简单的员工状态，后期要用three.js把员工转化为3d模型。时间流逝是按照游戏中的1分钟等于现实世界的2秒钟，然后可以有一个2倍加速。

### 一些设定和创意点

### 玩法设定

1. 玩家可以接人力外包和项目外包
2. 人力外包的话就是安排员工坐在玩家的办公室
3. 接项目外包就是在规定时间完成项目后拿钱，如果在规定时间没有完成项目则赔偿违约金
4. 玩家有一定的初始资金，比如5000快


### 人力外包设定

1. 甲方公司发布人员要求，和甲方给的每天预算，甲方分为急召和普通招聘，急召的话甲方给的预算会更多，但是限制在明天必须招到人，普通的话就是两天限制在后天结束前必须招到人，玩家可以选择接受合作。
2. 合作后如果在甲方要求的期限内没有找到合适的员工安排到甲方工作，那么需要每天支付给甲方每天的预算作为违约金
3. 甲方对于员工有一个产出要求（合作信息上会显示初级，中级，高级），如果员工的开发产出不满足，那么甲方会终止合作，终止之前会给玩家发邮件，给玩家1天时间再去寻找合适的员工
4. 这个产出要求给玩家显示为满意度，员工的摸鱼表现会导致产出降低，满意度下降

### 项目外包设定

1. 甲方发布项目外包，属性有完成时间，金额，如果未完成违约金。需要多少人力，分为产品，设计，前端后端开发，测试
2. 开发效率就是技能能力值/100，比如说他的技能能力值是80，那么他的开发效率就是80%
4. 项目延期要每天支付违约金
5. 项目要按照产品，设计，前后端开发，测试，四个步骤阶段进行。不能跳跃步骤


### 招聘设定

1. 玩家可以每天刷新3次简历，每次简历推荐5个候选人，也可以发布招聘公告，招聘公告上写工资和和技能条件，最多发3条，可能有人投敌简历，我这边也可以发offer，可以选择开通vip，每天可以刷新10次简历，开通vip后发布招聘公告收到的简历会翻5倍。也就是说可以浏览全部简历，也可以浏览投递的简历，投递的简历，如果按照公告上写的薪资发放offer那么大概率候选人会同意
2. 简历上有 头像（这个可以之后再做），毕业院校(普本，211，985和qs100)，工作经历，如果是刚毕业的工作经历就写无，然后个人技术简介(初级，中级，高级)+(产品，设计，前端和后端，后端，测试)，简历不对应员工的实际能力，但是能力高的员工大概率会写高级，能力低的员工大概率会写初级。然后如果员工工作年限长那么大概率也会有高级技能。
3. 员工的工资要求，员工的工资要求是按照天算的，如果员工的工作年限越长那么要求的工资也越高，如果员工的写的简历中技术也是高级那么工资要求也越高
4. 看到合适员工简历可以发offer，offer中可以填写每天的工资和社保基数，如果员工工资接近或者超过员工期望薪资那么员工就越有可能答应(综合判断社保基数)，如果员工答应了，那么就会招聘来

#### 员工设定

**玩家可以给新招聘的员工起花名**
**员工有技能属性，技能分为产品，设计，前端和后端，后端，测试，一个人可以有多个技能，但是大部分人都是一个技能，如果一个员工同时会前端和后端，那么他就是全栈开发者**
**员工工资分为税前工资，还可以调整员工的社保基数(选择工资的百分比再乘38%,提示包括养老保险，失业保险，医疗保险，工伤保险，生育保险，公积金)**
**员工有隐藏属性，分别为满意度，还有一个固定的仲裁倾向，如果员工满意度降低而工资社保没有满足法律规定(社保基数需要是100%)或者辞退员工没有给足赔偿(至少是n)那么员工可能会去起诉，摸鱼倾向，摸鱼倾向高的员工倾向在上班时间摸鱼，摸鱼的时候工作产出为0，工作能力值，如果是会多个技能的员工往往能力不高，还有一个隐藏属性是工作年限，如果工作年限越多那么仲裁倾向越高**
**员工有毕业院校,分为普本，211，985和qs100,普本，211，985和高能力值出现的可能增大，而且仲裁数值低，满意度高，qs100就是平均随机**
**员工在努力工作时，接的项目外包进度增加，在摸鱼时项目进度不变。不论是是否在摸鱼人力外包都是固定获取收益，但是如果摸鱼过多甲方的满意度会下降，下降到一定水平那么人力外包甲方会提出换人辞退他**
**辞退员工会提示赔偿，赔偿标准是按照中国法律规定，但是玩家可以自己调整比例, 展示为员工日工资*30*n*玩家设定的系数,默认系数是1，玩家可以最大设置为2，也可以设置为0，或者0.x,1.x都可以**

### 电子邮件系统：
签约、违约、合同预警、员工仲裁、仲裁结果、社保公积金投诉都会发邮件。
社保公积金投诉邮件发出时，立即扣除欠缴金额的 2 倍。
仲裁发起后第 3 天通过邮件通知结果，并执行赔偿或驳回。
